# 자유게시판
- 정해진 주제 없이 사람들끼리 자유롭게 대화를 나눌 수 있는 웹 사이트 게시판<br>
- 프로젝트 기간: 2020.03.02 ~ 2020.03.19<br>
- 목적
1. Node.js기초를 다지고 응용해보면서 코드 작성에 익숙해지기 위함
2. 최대한 간결하고 보기 좋은 코드를 작성하는 연습을 위함

## 사용 모듈
1. alert-node
2. body-parser
3. cookie-parser
4. cors
5. ejs
6. express
7. express-session
8. fs
9. http
10. moment
11. multer
12. mysql
13. path
14. pg
15. serve-static
16. session-file-store
